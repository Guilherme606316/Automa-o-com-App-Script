# Automação com Google Apps Script

## Descrição

Este repositório tem como objetivo criar automações e otimizações para tarefas utilizando o **Google Apps Script**.  
O Google Apps Script é uma ferramenta da Google que permite integrar código **JavaScript**, **HTML** e **CSS** aos serviços e produtos da Google, como Planilhas, Documentos, Gmail, entre outros.

## Características

- Automatização de tarefas rotineiras em serviços Google
- Integração fácil com Google Planilhas, Docs, Gmail, Slides, etc.
- Lógica programável usando JavaScript
- Personalização visual usando HTML e CSS
- Gratuito e acessível para qualquer usuário Google
- Permissões configuráveis (exemplo: envio de e-mails)
- Expansível para diversos setores e tipos de processos

## Pré-requisitos

- Conta Google (pode ser pessoal)
- Escolher a aplicação Google que deseja automatizar, como:
  - Google Planilhas
  - Google Docs
  - Gmail
  - Google Slides
  - Entre outros
- Conceder as permissões necessárias para o Apps Script realizar determinadas ações (por exemplo, enviar e-mails exige permissão específica)

## Modo de Usar

1. Escolha a aplicação Google que deseja automatizar (exemplo: Google Planilhas, Docs, Gmail, etc.).
2. Acesse a aplicação desejada.
3. No menu superior, vá em **Extensões** > **App Script**.
4. Você será redirecionado ao ambiente do App Script.
5. No App Script, crie seus códigos para automações e otimizações utilizando JavaScript para lógica, HTML para formato visual e CSS para personalização.
6. Execute e teste sua automação diretamente na aplicação escolhida.

> **Observação:**  
> O Google Apps Script executa ações predominantemente na aplicação escolhida, aproveitando sua integração direta com os serviços da Google.
